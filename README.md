# AI-Algorithms

This repository contains the lab experiments of AI ALgorithms Lab.

Experiments include:

* Breadth First Search

* Depth First Search

* A star Algorithm

* Adversarial Search

* Minimax Algorithm

* N Queen's Algorithm

* Tower of Hanoi

* Backtracking for CSP

* Local Search algorithms for CSP

* Travelling Salesman Problem

* Water sort puzzle
